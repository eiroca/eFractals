# eFractals

Programs and library to draw fractals. Program can draw trees, landscape, DLA, ...

* OS         : Win32 / MacOS X
* Type       : GUI
* Language   : FreePascal 3.0.x / Lazarus 1.8.x
* Dependency : -
* Version    : 1.1.0

[More information at official site](https://www.eiroca.net/efractals)
